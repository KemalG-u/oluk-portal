### Time to Value (TTV)

```
HEDEF: <5 dakika içinde değer hissi

1. Kayıt süreci basitleştirme (3 adım max)
2. Anında içerik erişimi (bekleme yok)
3. 2 dk mini-seans ile hızlı deneyim
4. Sırdaş ile kişisel bağlantı
```

## RETENTION LOOP'LARI

### Günlük Loop

```
Trigger: Sabah bildirimi (kullanıcı saatine göre)
     ↓
Action: App açma
     ↓
Reward: Günün afirmasyonu + yeni içerik
     ↓
Investment: Streak artışı, not alma
     ↓
[Sonraki gün için trigger]
```

### Haftalık Loop

```
Trigger: Pazar özet emaili
     ↓
Action: Haftalık ilerleme kontrolü
     ↓
Reward: Başarı rozeti, milestone
     ↓
Investment: Sonraki hafta hedef belirleme
     ↓
[Sonraki hafta için trigger]
```

### Sosyal Loop

```
Trigger: Arkadaş daveti / paylaşım
     ↓
Action: Yeni kullanıcı kaydı
     ↓
Reward: Her iki tarafa bonus (premium gün, rozet)
     ↓
Investment: İkili hesap verebilirlik
     ↓
[Yeni kullanıcı da davet eder]
```

## CONVERSİON OPTİMİZASYONU

### Free → Premium Funnel

```
AWARENESS (Farkındalık)
└─ Premium özellikleri gör

INTEREST (İlgi)
└─ Kısıtlı içeriğe tıkla, paywall gör

CONSIDERATION (Değerlendirme)
└─ Fiyatlandırma sayfası ziyareti

INTENT (Niyet)
└─ "Dene" butonuna tıkla

EVALUATION (Değerlendirme)
└─ 7 gün deneme başlat

PURCHASE (Satın alma)
└─ Ödeme yap
```

### Premium Değer Gösterimi

```
SOFT PAYWALL (Nazik):
"Bu içerik premium üyelere özel.
 7 gün ücretsiz dene, kaybedecek bir şey yok."

DEĞER ODAKLI:
"Premium ile bu ay:
 - 45 meditasyon izledin (değer: 450₺)
 - 21 gün streak tuttun
 - Sırdaş ile 23 sohbet ettin"

FOMO (Dikkatli kullan):
"Bu hafta 847 kişi premium'a geçti."
```

### A/B Test Alanları

```
1. Paywall tasarımı
2. Fiyatlandırma gösterimi (aylık vs yıllık)
3. CTA metni
4. Deneme süresi (7 gün vs 14 gün)
5. Onboarding akışı
6. Push notification içeriği
7. Email subject line
```

## REFERRAL PROGRAMI

### Mekanikler

```
DAVET EDEN:
- Her başarılı davet = 7 gün premium
- 5 davet = 1 ay premium
- 10 davet = "Topluluk Lideri" rozeti

DAVET EDİLEN:
- Kayıtta 7 gün premium deneme
- Özel hoş geldin içeriği

VİRALİTY HOOK:
- Kolay paylaşım linki
- Sosyal medya entegrasyonu
- İlerleme takibi (kaç davet, kaç kayıt)
```

### K-Factor Hesaplama

```
K = i × c

i: Ortalama davet sayısı (kullanıcı başına)
c: Davetin conversion oranı

HEDEF: K > 0.5 (her 2 kullanıcı 1 yeni getirir)
İDEAL: K > 1 (viral büyüme)
```

## DENEY FRAMEWORK'Ü

### ICE Skorlama

```
Impact (1-10): Ne kadar etki yaratır?
Confidence (1-10): Ne kadar eminiz?
Ease (1-10): Ne kadar kolay uygulanır?

ICE SKORU = (I + C + E) / 3

ÖNCELIK:
>7: Hemen yap
5-7: Plana al
<5: Backlog
```
