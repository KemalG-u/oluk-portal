## SEO STRATEJİSİ

### Anahtar Kelime Araştırması

```
BİRİNCİL ANAHTAR KELİMELER (Yüksek hacim):
├── meditasyon uygulaması
├── nefes egzersizi
├── stres yönetimi
├── içsel huzur
├── mindfulness türkçe
└── kişisel gelişim uygulaması

İKİNCİL ANAHTAR KELİMELER (Orta hacim):
├── sabah meditasyonu
├── uyku meditasyonu
├── kaygı için nefes teknikleri
├── tasavvuf meditasyon
├── sufi geleneği
└── 21 gün meditasyon

UZUN KUYRUK (Düşük hacim, yüksek niyet):
├── "iş yerinde stres nasıl azaltılır"
├── "gece uyuyamıyorum ne yapmalıyım"
├── "öfke kontrolü için meditasyon"
├── "türkçe rehberli meditasyon"
└── "spiritüel gelişim için uygulama"
```

### On-Page SEO Checklist

```
□ Başlık etiketi (H1): Anahtar kelime + çekici
□ Meta açıklama: 155 karakter, CTA içerir
□ URL yapısı: Kısa, Türkçe karaktersiz
□ H2-H3 başlıklar: Hiyerarşik, kelime içerir
□ Görsel alt text: Açıklayıcı
□ İç linkler: İlgili sayfalara 2-3 link
□ Dış linkler: Güvenilir kaynaklara
□ Kelime yoğunluğu: %1-2 (doğal)
□ Mobil uyumluluk: Responsive
□ Sayfa hızı: <3 saniye yükleme
```

### Teknik SEO

```
robots.txt: Doğru yapılandırma
sitemap.xml: Güncel ve submit edilmiş
Schema markup: Article, FAQ, HowTo
Canonical URL'ler: Duplicate önleme
HTTPS: Aktif
Core Web Vitals: Hedeflere uygun
Mobile-first indexing: Hazır
```

## BLOG STRATEJİSİ

### İçerik Türleri

```
1. EVERGREEN (Zamansız)
   - "Meditasyona Nasıl Başlanır: Başlangıç Rehberi"
   - "7 Temel Nefes Tekniği"
   - "Stres Yönetimi: Bilmeniz Gereken Her Şey"
   Amaç: Organik trafik, SEO

2. TREND/GÜNCEL
   - "2025'te Wellness Trendleri"
   - "Ramazan'da Meditasyon Pratiği"
   Amaç: Sosyal paylaşım, güncellik

3. DERİN DALıŞ
   - "Tasavvuf Geleneğinde Nefes"
   - "Polivagal Teori ve Meditasyon"
   Amaç: Otorite, uzman konumlandırma

4. LİSTE/KAYNAK
   - "En İyi 10 Sabah Rutini"
   - "Uyku İçin 5 Meditasyon"
   Amaç: Backlink çekme, paylaşım

5. KİŞİSEL HİKAYE
   - Kullanıcı başarı hikayeleri
   - Kurucu hikayesi
   Amaç: Güven, bağlantı
```

## EMAIL MARKETING

### Liste Segmentasyonu

```
SEGMENT 1: Yeni kayıtlar (0-7 gün)
→ Hoş geldin serisi

SEGMENT 2: Aktif kullanıcılar (haftalık aktif)
→ Haftalık özet, yeni içerik

SEGMENT 3: Pasif kullanıcılar (14+ gün inaktif)
→ Re-engagement kampanyası

SEGMENT 4: Premium kullanıcılar
→ Özel içerik, erken erişim

SEGMENT 5: Churn risk (30+ gün inaktif)
→ Win-back kampanyası
```

### Email Türleri

```
1. HOŞ GELDİN SERİSİ (Otomatik)
   Gün 1: Hoş geldin + ilk adım
   Gün 3: En popüler içerik
   Gün 5: Sırdaş tanıtımı
   Gün 7: Topluluk daveti

2. HAFTALIK BÜLTEN
   - Haftanın afirmasyonu
   - Yeni içerik özeti
   - Topluluktan haberler
   - Bir ipucu/pratik

3. KAMPANYA EMAİLLERİ
   - Yeni özellik duyurusu
   - Bayram/özel gün
   - Premium promosyon

4. İŞLEMSEL EMAİLLER
   - Şifre sıfırlama
   - Ödeme onay
   - Streak hatırlatma
```

### Email Best Practices

```
KONU SATIRI:
- 40-50 karakter
- Kişiselleştirme ([name])
- Emoji: max 1
- A/B test

İÇERİK:
- Mobile-first tasarım
- Tek CTA odak
- Alt text görsellerde
- Unsubscribe kolay

GÖNDERİM:
- Salı-Perşembe optimal
- 10:00 veya 14:00
- Spam kontrolü
```
