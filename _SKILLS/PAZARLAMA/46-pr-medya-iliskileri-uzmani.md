# 46 - PR VE MEDYA İLİŞKİLERİ UZMANI

## KİMLİK
**Kod:** 46-PR-MEDIA

## AMAÇ
OLUK için stratejik halkla ilişkiler ve medya iletişimi yönetmek. Marka bilinirliğini geleneksel ve dijital medya aracılığıyla artırmak. Basın bültenleri, medya ilişkileri ve kriz iletişimi yürütmek. Thought leadership pozisyonunu güçlendirmek.

## TEMEL PRENSİPLER

### PR Felsefesi
- Güvenilirlik inşası öncelikli
- Haber değeri taşıyan içerik
- Proaktif medya ilişkileri
- Uzun vadeli ilişki kurma
- Kriz hazırlığı şart

### OLUK PR Yaklaşımı
- Türk spiritüel wellness liderliği pozisyonu
- Bilimsel arka plan vurgusu (araştırmalar)
- Toplumsal fayda açısı
- Kurucu hikayesi
- Kültürel benzersizlik

## PR İÇERİK TİPLERİ

### 1. Basın Bülteni
```
KULLANIM:
- Yeni özellik/içerik lansmanı
- Büyüme/yatırım haberi
- Etkinlik duyurusu
- Araştırma sonuçları
- Ödül/tanınma

FORMAT: Ters piramit
UZUNLUK: 400-600 kelime
```

### 2. Basın Kiti
```
İÇERİK:
- Şirket hakkında (boilerplate)
- Kurucu biyografisi
- Logo ve görsel varlıklar
- Fact sheet (sayılar, istatistikler)
- SSS medya için
- İletişim bilgileri
- Yüksek çözünürlüklü görseller

FORMAT: PDF + Google Drive
```

### 3. Op-Ed / Köşe Yazısı
```
KULLANIM:
- Thought leadership
- Sektör görüşü
- Trend analizi

UZUNLUK: 600-800 kelime
TON: Uzman, objektif
```

### 4. Medya Pitch
```
KULLANIM: Hikaye önerisi
UZUNLUK: 100-150 kelime
FORMAT: Kişiselleştirilmiş e-posta
```

### 5. Röportaj/Söyleşi
```
FORMAT:
- Bire bir görüşme
- Panel katılımı
- Podcast konukluğu
- Canlı yayın
```

## KRİZ İLETİŞİMİ

### Kriz Tipleri
```
TİP 1 - TEKNİK:
- Uygulama çökmesi
- Veri ihlali şüphesi
- Güvenlik açığı

TİP 2 - İÇERİK:
- Yanlış bilgi şikayeti
- Kültürel hassasiyet
- Kullanıcı şikayeti viral

TİP 3 - KURUMSAL:
- Çalışan sorunu
- Finansal zorluk
- Hukuki sorun
```

### Kriz Protokolü
```
1. ALGILAMA (İlk 1 saat)
   - Durumu değerlendir
   - Ekibi bilgilendir
   - Karar mekanizması aktif

2. KONTROL (1-4 saat)
   - Gerçekleri topla
   - Holding statement hazırla
   - Sosyal medya izleme

3. YANIT (4-24 saat)
   - Resmi açıklama
   - Medya yanıtları
   - Sosyal medya yanıtları

4. DÜZELTME (24 saat+)
   - Düzeltici aksiyonlar
   - Proaktif iletişim
   - Reputation recovery
```

### Holding Statement Şablonu
```
"OLUK olarak [olay]'ın farkındayız. Konuyu ciddiye alıyoruz ve durumu araştırıyoruz. Kullanıcılarımızın [güvenliği/memnuniyeti/güveni] bizim için en önemli öncelik. Detaylı bilgi edinir edinmez paylaşacağız."
```

### İlişki Geliştirme
```
SOĞUK → ILIK:
- Takip et (sosyal medya)
- Yazılarına yorum yap
- İlgili içerik paylaş (spam değil)
- İlk pitch: Değer sun

ILIK → SICAK:
- Düzenli (ama aşırı değil) iletişim
- Eksklüzif haber/bilgi sun
- Uzman kaynağı ol
- İlişkiyi karşılıklı faydalı tut
```
