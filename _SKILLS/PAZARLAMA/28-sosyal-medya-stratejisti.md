## PLATFORM STRATEJİLERİ

### Instagram (Ana Platform)

```
HEDEF KİTLE: 25-45 yaş kadın, şehirli, eğitimli
İÇERİK MİKSİ:
├── %40 Afirmasyon kartları (carousel)
├── %25 Eğitici içerik (bilgi grafikleri)
├── %20 Story (günlük, behind the scenes)
├── %10 Reels (kısa meditasyon, nefes)
└── %5 IGTV/Video (uzun form)

POST SIKLIĞI: 
- Feed: Günde 1
- Story: Günde 3-5
- Reels: Haftada 3

EN İYİ SAATLER:
- Hafta içi: 07:00-09:00, 12:00-14:00, 19:00-21:00
- Hafta sonu: 10:00-12:00

HASHTAG STRATEJİSİ:
Marka: #OLUK #OLUKile #AkışınKanalı
Niş: #içselhuzur #nefesal #türkmeditasyon
Genel: #kişiselgelişim #mindfulness #wellness
```

### YouTube

```
HEDEF KİTLE: Derinleşmek isteyen, araştırmacı
İÇERİK MİKSİ:
├── Rehberli meditasyonlar (10-30 dk)
├── Eğitim videoları (konsept açıklamaları)
├── Shorts (60 sn nefes/afirmasyon)
└── Podcast formatı (sohbetler)

POST SIKLIĞI:
- Uzun video: Haftada 1
- Shorts: Haftada 3-5

SEO:
- Başlıkta anahtar kelime
- Açıklama ilk 150 karakter önemli
- Türkçe altyazı şart
- Özel thumbnail
```

### TikTok

```
HEDEF KİTLE: 18-30 yaş, keşfetmeye açık
İÇERİK MİKSİ:
├── Hızlı nefes teknikleri
├── Afirmasyon okuma
├── "POV" formatında içerikler
├── Trend seslere adaptasyon
└── Duet/Stitch imkanları

POST SIKLIĞI: Günde 1-2

DİKKAT:
- Platform doğasına uyum (eğlenceli ama özlü)
- Trend takibi ama marka bütünlüğü
- İlk 3 saniye kritik
```

### Twitter/X

```
HEDEF KİTLE: Fikir liderleri, profesyoneller
İÇERİK MİKSİ:
├── Günlük afirmasyon (sabah)
├── Thread'ler (eğitici içerik)
├── Alıntılar (kitaplardan, ustalardan)
├── Topluluk etkileşimi
└── Sektör haberleri/yorumlar

POST SIKLIĞI: Günde 2-3

THREAD KONULARI:
- "7 günde stres yönetimi"
- "Meditasyona başlayanlar için rehber"
- "Nefes tekniklerinin bilimi"
```

### LinkedIn

```
HEDEF KİTLE: Kurumsal wellness, HR, liderler
İÇERİK MİKSİ:
├── Thought leadership makaleler
├── İş yerinde mindfulness
├── Kurucu hikayesi
├── Şirket güncellemeleri
└── İş ilanları

POST SIKLIĞI: Haftada 2-3

TON: Profesyonel ama sıcak
```

### Carousel Eğitim (Instagram)

```
Slide 1: Kapak (dikkat çekici başlık)
Slide 2-7: İçerik (1 fikir per slide)
Slide 8: CTA ("Kaydet", "Arkadaşına gönder")
Slide 9: OLUK tanıtım (opsiyonel)

BOYUT: 1080x1350 (dikey tercih)
MAX SLIDE: 10
```

### Reels Şablonu

```
0-3 sn: Hook (dikkat çek)
3-15 sn: Değer ver (bilgi/pratik)
15-30 sn: CTA veya tekrar
30-60 sn: Uzun form (gerekirse)

MÜZİK: Sakin, telif ücretsiz
ALTYAZI: Şart (sessiz izleyenler)
```

## TOPLULUK YÖNETİMİ

### Yorum Yanıt Protokolü

```
YANIT SÜRESİ:
- DM: 24 saat içinde
- Yorum: 12 saat içinde
- Mention: 6 saat içinde

YANIT TONU:
- Kişiselleştirilmiş (isim kullan)
- Sıcak ve destekleyici
- Emoji: Minimal (1-2)
- Satış yok
```

### Kriz Yönetimi

```
SEVİYE 1: Basit şikayet
→ Özür + çözüm önerisi

SEVİYE 2: Ciddi eleştiri
→ DM'e taşı + yönetici bilgilendir

SEVİYE 3: Viral negatif
→ Resmi açıklama + PR ekibi
```

### UGC (Kullanıcı İçeriği)

```
PAYLAŞIM KURALLARI:
1. İzin al (DM veya yorum)
2. Kredi ver (@mention)
3. Teşekkür et
4. Story > Feed (daha samimi)
```
