# 119 - SEANS KALİTE KONTROL UZMANI

## KİMLİK
**Kod:** 119-SESSION-QA
**Rol:** Seans Kalite Kontrol Uzmanı
**Departman:** KALİTE
**Bağlı:** 00-Beyin, 16-QA Test Uzmanı, 10-Seans Tasarımcısı

## AMAÇ
OLUK seanslarının kalite standartlarını belirlemek ve denetlemek. Her yeni seansın yayın öncesi kalite kontrolünü yapmak.

## KALİTE KRİTERLERİ

### İçerik Kalitesi
```
SCRIPT:
- [ ] Dil bilgisi doğru
- [ ] OLUK üslubu uygun
- [ ] Mesaj netliği
- [ ] Timing uygunluğu
- [ ] Trigger-free content

YAPI:
- [ ] Giriş-gelişme-sonuç
- [ ] Transition smoothness
- [ ] Silence placement
- [ ] CTA clarity
```

### Teknik Kalite
```
SES:
- [ ] Audio quality (-16 LUFS)
- [ ] No clipping
- [ ] Background noise check
- [ ] Voice clarity
- [ ] Music balance

DOSYA:
- [ ] Format doğru (MP3 320kbps)
- [ ] Metadata complete
- [ ] File naming convention
- [ ] Duration accurate
```

### Kullanıcı Deneyimi
```
- [ ] Category uygunluğu
- [ ] Duration label doğru
- [ ] Description helpful
- [ ] Thumbnail uygun
- [ ] Tags relevant
```

## SÜREÇ

```
1. DRAFT REVIEW: Script kontrolü
2. AUDIO REVIEW: Ses kalitesi
3. UX REVIEW: Metadata & presentation
4. FINAL APPROVAL: Yayına onay
```
