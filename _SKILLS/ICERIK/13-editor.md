## UZMANLIK ALANLARI

### 1. YAPI EDİTÖRLÜĞÜ

#### Mantıksal Akış Kontrolü

```
KONTROL LİSTESİ:

□ Giriş konuyu tanıtıyor mu?
□ Her bölüm bir öncekine bağlı mı?
□ Geçişler doğal mı?
□ Sonuç başa dönüyor mu?
□ "Havada kalan" bölüm var mı?
□ Sıralama mantıklı mı?
```

#### Bölüm Dengesi

```
İDEAL DENGE:

DERS (15 dk):
- Giriş: %15 (2 dk)
- Ana içerik: %60 (9 dk)
- Etkileşim: %10 (1.5 dk)
- Kapanış: %15 (2.5 dk)

SEANS (15 dk):
- Hazırlama: %20 (3 dk)
- Derinleşme: %20 (3 dk)
- Çalışma: %40 (6 dk)
- Entegrasyon: %10 (1.5 dk)
- Dönüş: %10 (1.5 dk)

Bir bölüm çok uzun/kısa → Dengele
```

#### Eksik/Fazla Tespiti

```
EKSIK OLABILIR:
- Örnek (kavram anlatıldı ama örnek yok)
- Geçiş (iki bölüm arası atlama)
- Motivasyon ("neden önemli" eksik)
- Özet (ana mesaj kaybolmuş)
- Köprü (seans/egzersiz bağlantısı yok)

FAZLA OLABILIR:
- Tekrar (aynı şey iki kez söylenmiş)
- Ayrıntı (gereksiz detay)
- Kavram (bir içerikte çok kavram)
- Açıklama (zaten anlaşılmış, uzatılmış)
```

### 2. TUTARLILIK KONTROLÜ

#### İç Tutarlılık

```
KONTROL:
□ Aynı kavram her yerde aynı mı?
□ Çelişki var mı?
□ Ton baştan sona tutarlı mı?
□ "Sen/siz" karışıklığı var mı?
□ Zaman kipi tutarlı mı?
```

#### Platform Tutarlılığı

```
KONTROL:
□ OLUK terminolojisi kullanılmış mı?
□ Faz yapısına uygun mu?
□ Önceki içeriklerle çelişiyor mu?
□ Platform sözlüğüne uygun mu?
```

#### Örnek: Tutarsızlık Tespiti

```
SORUN:
Ders başında: "Nefes, farkındalığın kapısıdır"
Ders ortasında: "Nefes, sakinliğin anahtarıdır"
→ Metafor değişmiş (kapı → anahtar)

ÇÖZÜM:
Tek metaforu seç ve tüm metinde kullan
VEYA
Bilinçli geçiş yap: "Kapı açıldı, şimdi anahtar sende"
```

### 3. ANLAŞILIRLIK KONTROLÜ

#### Hedef Kitle Uyumu

```
OLUK KİTLESİ:
- %95 kadın
- Çoğu 25-45 yaş
- Çoğu üniversite mezunu
- Spiritüel meraklı ama yeni başlayan
- Mobil kullanıcı

KONTROL:
□ Jargon açıklanmış mı?
□ Cümleler çok uzun mu?
□ Soyut kavramlar somutlaştırılmış mı?
□ Örnekler hedef kitleye uygun mu?
```

#### Karmaşıklık Seviyesi

```
KURAL: Her içeriğin hedef fazına uygun olmalı

FAZ 1: Basit, temel kavramlar
FAZ 2: Orta, önceki kavramları kullanabilir
FAZ 3: İleri, karmaşık bağlantılar kurabilir
FAZ 4: Derin, felsefik derinlik olabilir

FAZ 1 içeriğinde FAZ 3 karmaşıklığı = SORUN
```

---

### 4. EDİTÖRYAL İŞARETLER

#### İşaretleme Sistemi

```
[?] = Anlaşılmadı, açıklama gerekiyor
[!] = Dikkat, potansiyel sorun
[+] = Eksik, eklenmeli
[-] = Fazla, çıkarılmalı
[~] = Değişmeli, öneride bulun
[→] = Geçiş sorunu
[≠] = Tutarsızlık
[✓] = İyi, kalsın
```

#### Örnek Kullanım

```
Orijinal metin:
"Nefes hayatın temelidir. Nefes almadan yaşayamayız. [!tekrar]
Şimdi gözlerini kapat. [-klişe]
Derin bir nefes al. [?kaç saniye]
Fark et. [+neyi fark edecek]"

Editör notu:
- İlk iki cümle aynı şeyi söylüyor, birini çıkar
- "Gözlerini kapat" klişe, alternatif öner
- Nefes süresi belirsiz, netleştir
- "Fark et" çok belirsiz, somutlaştır
```
