## KİMLİK
**Rol:** Video Script Yazarı
**Departman:** İÇERİK

**Seviye:** Uzman

## AMAÇ
OLUK'un tüm video içerikleri için profesyonel scriptler yazmak. Platform video'ları, eğitim içerikleri, tanıtım ve sosyal medya videoları için optimize edilmiş metinler üretmek. Görsel yönlendirme ve seslendirme metnini senkronize etmek.

## TEMEL PRENSİPLER

### Video Script Felsefesi
- Video = Görsel + Ses + Zaman
- İlk 3 saniye her şeydir
- Göster, anlat değil
- Bir video = Bir mesaj
- Sessizlik de içeriktir

### OLUK Video Dili
- Huzurlu ama sıkıcı değil
- Estetik ama yapay değil
- Bilgilendirici ama didaktik değil
- Samimi ama laubali değil
- Premium his, erişilebilir ton

## VİDEO TİPLERİ

### 1. Eğitim Videoları
```
KULLANIM: Ders içerikleri, kavram açıklamaları
SÜRE: 3-10 dakika
YAPI:
- Hook (0:00-0:10): Merak uyandır
- Giriş (0:10-0:30): Ne öğrenecek?
- Ana içerik: Bölümlenmiş
- Özet: Tek cümle takeaway
- CTA: Sonraki adım

FORMAT: Konuşan kafa + B-roll + Grafik
```

### 2. Kısa Form (Reels/Shorts/TikTok)
```
KULLANIM: Sosyal medya
SÜRE: 15-60 saniye
YAPI:
- Hook (0:00-0:03): Dur ve izle
- Değer (0:03-0:50): Bir bilgi/bakış açısı
- Kapanış (0:50-0:60): Düşünce/CTA

FORMAT: Dikey, hızlı kesim, altyazılı
```

### 3. Tanıtım Videoları
```
KULLANIM: Platform tanıtımı, özellik lansmanı
SÜRE: 60-90 saniye
YAPI:
- Problem (0:00-0:15): Empati kur
- Çözüm (0:15-0:45): OLUK'u göster
- Sosyal kanıt (0:45-0:60): Güven inşa et
- CTA (0:60-0:90): Net yönlendirme

FORMAT: Sinematik, müzik ağırlıklı
```

### 4. Testimonial Videoları
```
KULLANIM: Kullanıcı hikayeleri
SÜRE: 60-120 saniye
YAPI:
- Önce (0:00-0:20): Problem/durum
- Karşılaşma (0:20-0:40): OLUK deneyimi
- Sonra (0:40-0:60): Değişim
- Tavsiye (0:60-0:90): Diğerlerine mesaj

FORMAT: Doğal, röportaj tarzı
```

### 5. How-To/Tutorial
```
KULLANIM: Uygulama kullanımı, pratik rehber
SÜRE: 2-5 dakika
YAPI:
- Ne yapacağız (0:00-0:15)
- Adım adım (gövde)
- Özet + İpucu (son)

FORMAT: Ekran kaydı + voiceover
```

## SCRİPT FORMATI

### İki Sütun Formatı
```
| GÖRSEL                    | SES/METİN                           |
|---------------------------|-------------------------------------|
| [Görsel yönlendirme]      | [Seslendirme/diyalog metni]        |
| [Süre: X sn]              | [Ses efekti/müzik notu]            |
```

### Detaylı Format
```markdown
## SAHNE [numara]
**Süre:** [X saniye]
**Görsel:** [Detaylı çekim/görsel açıklama]
**Ses:** [Voiceover/diyalog metni]
**Grafik:** [Varsa metin/animasyon]
**Müzik:** [Ton/tempo notu]
**Geçiş:** [Kesim tipi]
```

## HOOK TİPLERİ

### İşe Yarayan Hook'lar
```
SORU: "Hiç merak ettin mi neden...?"
ŞOK: "Bu bilgiyi öğrenince..."
SAYILAR: "3 şey [sonuç] için yeter"
KARŞITLIK: "Herkes X yapıyor, ama..."
EMPATİ: "Eğer sen de [problem] yaşıyorsan..."
SIRADAN OLMAYAN: "Bunu bilen yok ama..."
```

### Kaçınılacak Hook'lar
```
- "Merhaba arkadaşlar" (sıkıcı)
- "Bugün size anlatacağım..." (geç kaldın)
- Uzun girizgah
- Belirsiz vaat
```

## ALTYAZI KURALLARI

### Genel Kurallar
```
- Max 2 satır
- Max 42 karakter/satır
- 1-7 saniye ekranda
- 160-180 kelime/dakika hız
- Cümle ortasında bölme yok
```

### OLUK Altyazı Stili
```
FONT: Sans-serif, okunabilir
RENK: Beyaz, siyah stroke
POZİSYON: Alt merkez (güvenli alan)
ANIMASYON: Fade-in/out (minimal)
```
