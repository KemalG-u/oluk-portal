## SES SCRİPTİ YAZIM KURALLARI

### Dil ve Üslup

```
✓ KULLANILACAK:
- Kısa, net cümleler (max 15 kelime)
- Yumuşak geçişler ("Şimdi...", "Yavaşça...")
- Duyusal betimlemeler ("Hisset", "Fark et", "İzin ver")
- Davetkar dil ("...yapabilirsin", "...izin ver")
- Sessizlik/pause işaretleri [3 sn], [5 sn]
- Nefes hatırlatmaları

✗ KAÇINILACAK:
- Emir kipi ("Yap!", "Düşün!")
- Uzun, karmaşık cümleler
- Soyut kavramlar (açıklama olmadan)
- Negatif ifadeler ("Yapma", "Düşünme")
- Jargon veya teknik terimler
- Aceleci geçişler
```

### Pause (Sessizlik) Kullanımı

```
[2 sn]  → Cümle arası kısa duraklama
[3 sn]  → Talimat sonrası uygulama süresi
[5 sn]  → Nefes veya kısa pratik
[10 sn] → Deneyimleme/hissetme süresi
[20 sn] → Derin pratik bölümü
[30 sn] → Sessiz meditasyon kesiti
[1 dk]  → Uzun sessizlik (müzikle)
```

### Örnek Script Parçası

```markdown
## IŞIK MEDİTASYONU - Açılış Bölümü (2 dk)

Hoş geldin. [2 sn]

Şimdi rahat bir pozisyon bul.
Oturuyorsan sırtın dik, omuzların rahat olsun.
Uzanıyorsan kolların bedeninin yanında dinlensin. [3 sn]

Gözlerini yavaşça kapat. [2 sn]

Derin bir nefes al... [3 sn]
Ve yavaşça bırak. [3 sn]

Bir nefes daha... [3 sn]
Bırak. [3 sn]

Bu an sadece senin.
Dış dünyanın sesleri uzaklaşsın.
Yapılacaklar listesi bekleyebilir.
Şu an, burada, kendinle baş başasın. [5 sn]

Niyetini belirle.
Bu meditasyondan ne almak istiyorsun?
Bir kelime ya da bir his yeterli. [5 sn]

Hazırsan, ışığın yolculuğuna başlayalım.
```

## SES PRODÜKSIYON REHBERİ

### Teknik Gereksinimler

```
Format: MP3 (320 kbps) + WAV (arşiv)
Sample Rate: 44.1 kHz
Bit Depth: 16-bit
Mono/Stereo: Stereo (arka plan müziği varsa)
Normalizasyon: -14 LUFS (podcast standardı)
```

### Ses Katmanları

```
1. ANA SES (Rehber)
   - Net, sakin, sıcak ton
   - Yakın mikrofon (intimacy hissi)
   - Minimum oda yankısı

2. ARKA PLAN MÜZİĞİ
   - -20 dB (sesin altında)
   - Enstrümantal, vokal yok
   - Loop noktaları kesintisiz
   - Telif hakları temiz

3. DOĞA SESLERİ (Opsiyonel)
   - Su sesi, kuş, rüzgar
   - Çok hafif (-25 dB)
   - Dikkat dağıtmayan
```

### Müzik Önerileri

```
Ney (geleneksel, OLUK ruhu)
Ambient pad'ler
Singing bowl / Tibet çanakları
Doğa soundscape'leri
432 Hz frekans müzikleri
Binaural beats (opsiyonel)
```
