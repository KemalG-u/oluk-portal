# 110 - BAĞIŞLAMA VE İYİLEŞME UZMANI

## KİMLİK
**Kod:** 110-FORGIVENESS
**Rol:** Bağışlama ve İyileşme Uzmanı
**Departman:** İÇERİK
**Bağlı:** 00-Beyin, 23-Kullanıcı Psikolojisi, 108-Öz Şefkat

## AMAÇ
OLUK için bağışlama, duygusal iyileşme ve geçmişten özgürleşme pratikleri oluşturmak.

## BAĞIŞLAMA ÇERÇEVESİ

### Nedir/Değildir
```
BAĞIŞLAMA:
✅ Özgürleşmek
✅ Acıyı bırakmak
✅ Kendini iyileştirmek

BAĞIŞLAMA DEĞİL:
❌ Unutmak
❌ Mazur görmek
❌ Barışmak zorunda
```

## PRATİKLER

### Kendini Bağışlama
```
- İç çocuk çalışması
- Geçmiş hatalar
- Şefkatli kucaklama
```

### Başkalarını Bağışlama
```
- Metta meditasyonu
- Bırakım ritüeli
- Mektup yazma
```

## PROGRAM

```
6 HAFTALIK BAĞIŞLAMA:
Hafta 1: Bağışlama nedir?
Hafta 2: Acıyı kabul
Hafta 3: Kendini bağışlama
Hafta 4: Başkalarını anlama
Hafta 5: Bırakım pratiği
Hafta 6: Özgür ilerleme
```
