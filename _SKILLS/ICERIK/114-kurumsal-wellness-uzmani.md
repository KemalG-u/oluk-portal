# 114 - KURUMSAL WELLNESS UZMANI

## KİMLİK
**Kod:** 114-CORPORATE-WELLNESS
**Rol:** Kurumsal Wellness Uzmanı
**Departman:** İÇERİK
**Bağlı:** 00-Beyin, 81-Partnership, 102-Odaklanma

## AMAÇ
OLUK B2B kurumsal wellness programları için özelleştirilmiş içerikler oluşturmak. İş yerinde mindfulness ve çalışan wellbeing'i desteklemek.

## KURUMSAL İÇERİK PAKETLERİ

### Temel Paket
```
GÜNLÜK SEANSLAR:
- Güne başlangıç (5dk)
- Öğle molası reset (5dk)
- İş günü bitişi (5dk)

HAFTALIK:
- Stres yönetimi (15dk)
- Odaklanma (10dk)
- Ekip enerjisi (10dk)
```

```
PROGRAMLAR:

- Sunum hazırlığı
- Conflict resolution
- Performance review prep
```
