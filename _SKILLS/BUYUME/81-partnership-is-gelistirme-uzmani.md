## PARTNER DEĞERLENDİRME

### Uygunluk Kriterleri
```
MARKA UYUMU (1-5):
- Değerler örtüşüyor mu?
- Hedef kitle uyumlu mu?
- İmaj uyumlu mu?

REACH (1-5):
- Erişim büyüklüğü
- Hedef kitledeki penetrasyon
- Aktif kullanıcı/müşteri

KAYNAK (1-5):
- Ne katkı sağlayabilirler?
- Bizden ne bekliyorlar?
- Sürdürülebilir mi?

STRATEJİK DEĞER (1-5):
- Uzun vadeli potansiyel
- Öğrenme fırsatı
- Referans değeri

TOPLAM SKOR: /20
≥16: Öncelikli takip
12-15: Değerlendirmeye devam
<12: Şimdilik değil
```

## B2B SALES PROCESS

### Kurumsal Satış Hunisi
```
LEAD → MQL → SQL → OPPORTUNITY → PROPOSAL → CLOSE

LEAD:
- Inbound inquiry
- Outbound prospecting
- Event leads

MQL (Marketing Qualified):
- Şirket büyüklüğü uygun
- Sektör uygun
- İlgi gösterdi

SQL (Sales Qualified):
- Budget var
- Karar verici ile görüşüldü
- Timeline var

OPPORTUNITY:
- İhtiyaçlar belirlendi
- Çözüm sunuldu
- Aktif değerlendirme

PROPOSAL:
- Resmi teklif gönderildi
- Fiyatlandırma sunuldu
- Sözleşme taslağı

CLOSE:
- Sözleşme imzalı
- Onboarding başladı
```

### B2B Fiyatlandırma
```
KURUMSAL PAKETLERİ:

STARTER (10-50 çalışan):
- ₺50/kullanıcı/ay
- Temel raporlama
- Email destek

PROFESSIONAL (50-200 çalışan):
- ₺40/kullanıcı/ay
- Gelişmiş raporlama
- Dedicated CSM
- Özel içerik (sınırlı)

ENTERPRISE (200+ çalışan):
- Özel fiyatlandırma
- Full customization
- On-site training
- API access
- White-label seçeneği
```
