### 2. MÜFREDAT MİMARİSİ

#### Yapı Türleri

```
LİNEER:
A → B → C → D
✓ Basit, anlaşılır
✗ Esnek değil
→ Faz içi zorunlu sıralama için

MODÜLER:
    ┌─ B1 ─┐
A ──┼─ B2 ─┼── C
    └─ B3 ─┘
✓ Esnek, seçim imkanı
✗ Kaybolma riski
→ İleri fazlarda seçmeli için

HİBRİT (OLUK):
Faz 1-2: Lineer (temel oluştur)
Faz 3-4: Modüler (derinleş)
```

#### Ön Koşul Tasarımı

```
TİP                | AÇIKLAMA                    | ÖRNEK
-------------------|-----------------------------|--------------------------
Sert Ön Koşul      | Mutlaka tamamlanmalı        | Faz 2 için Faz 1 şart
Yumuşak Ön Koşul   | Önerilir, zorunlu değil     | "Önce X dersini öneririz"
Kavramsal Ön Koşul | Belirli kavramı bilmeli     | Nefs kavramını anlamak

DİKKAT: Her ders MAX 2 sert ön koşul
```

#### Zorluk Kademesi (Flow State için)

```
Çok kolay = Sıkılma
Çok zor = Kaygı
Tam kıvam = Akış

ZORLUK DAĞILIMI:
Seviye 1: Herkes yapabilir    → Faz 1 başı
Seviye 2: Çoğu yapabilir      → Faz 1-2
Seviye 3: Pratik gerekir      → Faz 2-3
Seviye 4: Deneyim gerekir     → Faz 3-4
Seviye 5: Ustalık gerekir     → Faz 4
```

### 4. İLERLEME VE MOTİVASYON TASARIMI

#### Milestone Sistemi

```
KÜÇÜK ZAFERLER:
- Her ders sonu: ✓ Tamamlandı
- Her seans sonu: Pratik puan +1
- Her modül sonu: Modül rozeti
- Her faz sonu: Faz sertifikası
```

#### Hook Model (Nir Eyal)

```
1. Tetikleyici → Bildirim, hatırlatma
2. Eylem      → Ders/seans başlat
3. Ödül       → Rozet, insight, topluluk
4. Yatırım    → Profil, ilerleme, streak
```

#### Kritik Retention Noktaları

```
GÜN    | HEDEF                      | AKSİYON
-------|----------------------------|---------------------------
1      | İlk deneyim mükemmel       | Kolay, etkileyici içerik
3      | İlk "aha" anı              | Insight verici ders
7      | İlk milestone              | Kutlama + rozet
14     | Modül tamamlama            | Başarı mesajı
30     | Faz 1 bitişe yakın         | Motivasyon desteği
60     | Faz 2 ortası               | Topluluk entegrasyonu
```

### 1. Yeni İçerik Konumlandırma

```
│ - Ön koşulları?               │

        ↓
┌───────────────────────────────┐
│ FAZ BELİRLEME                 │
│ - Temel farkındalık → Faz 1   │

│ - Derin/ustalık → Faz 4       │
└───────────────────────────────┘
        ↓
┌───────────────────────────────┐
│ MODÜL + SIRA BELİRLEME        │
│ - Hangi modüle uyuyor?        │
│ - Ön koşul dersleri?          │
│ - Modül içi mantıksal sıra?   │
└───────────────────────────────┘
        ↓
┌───────────────────────────────┐
│ POZİSYON KARTI ÇIKTI          │
│ Faz: X, Modül: Y, Sıra: Z     │
└───────────────────────────────┘
```

### 2. Müfredat Dengeleme Kontrol Listesi

```
HER FAZ İÇİN KONTROL:
□ Teori/Pratik dengesi: En az %40 seans
□ Ders uzunluğu: Ortalama 10-15 dakika
□ Seans uzunluğu: 5-20 dakika (çeşitli)
□ Modül başına ders: 3-5 arası
□ Checkpoint sıklığı: Her 2-3 modülde
□ Zorluk eğrisi: Kademeli artış
□ Tekrar noktaları: Anahtar kavramlar
□ Mola anları: Yoğun içerik sonrası
```

### 3. Tıkanma Noktası Analizi

```
İZLENECEK METRİKLER:
- Ders başlama ama tamamlamama oranı
- Seans tekrar oranı (çok tekrar = sorun)
- Modül tamamlama süreleri
- Faz geçiş oranları
- Geri dönüş noktaları

ALARM EŞİKLERİ:
- Ders tamamlama <%70 → İçerik sorunu
- Modül >=14 gün → Tıkanma
- Faz geçiş <%50 → Ciddi sorun

MÜDAHALE:
- İçeriği böl
- Ara ders ekle (köprü)
- Ön koşulu yumuşat
- Motivasyon içeriği güçlendir
- Alternatif yol aç
```
